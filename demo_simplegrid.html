Responsive Image Gallery

JQuery plugin to display a grid of images along with secondary thumbnail images displayed in a FancyBox. Numerous settings such as images per row (3, 4 or 6), images per page, two types of pagination (on scroll and via page buttons), captions can be displayed staticly under each image or in an overlay on hover, container can be set for any image dimension (using images of the same or similar dimension recommended), images can be fit to their containers or auto-cropped from the bottom up.
